# Edinburgh University Kendo Club Website

Static website for the Edinburgh University Kendo Club.

## Overview

This repository contains the club website built with plain HTML, CSS, and JavaScript.

- No framework
- No build step
- Suitable for free static hosting such as GitHub Pages

## Project Structure

- `index.html`: homepage
- `about.html`: club overview and training information
- `training-and-membership.html`: schedule, membership, visitor guidance, and location
- `committee.html`: committee page
- `contact.html`: contact page
- `assets/css/`: shared styles
- `assets/js/`: shared JavaScript and committee data
- `images/`: site images and committee photos
